Una de las preguntas más frecuentes que recibo es: *"Omar, necesito una landing page o un sitio web completo?"* La respuesta corta es: depende de tu objetivo de negocio. La respuesta larga está en este artículo.

## Diferencia real entre landing page y sitio web

### Landing page (página de aterrizaje)
- **Una sola página** con información focalizada
- **Un objetivo claro**: captar emails, vender un producto, agendar citas
- **Sin navegación compleja**: el usuario scrollea o hace clic en el CTA principal
- **Diseño persuasivo**: copy orientado a conversión, no a información general

### Sitio web completo
- **Múltiples páginas**: inicio, nosotros, servicios, blog, contacto, etc.
- **Navegación con menú**: usuarios pueden explorar diferentes secciones
- **Información completa**: toda la historia de tu empresa, portafolio, equipo
- **SEO extenso**: más páginas = más oportunidades de aparecer en Google

## Cuándo elegir landing page

### Elegir landing page si:

**1. Vas a lanzar un producto o servicio específico**
Ejemplo: Eres un dentista en Bogotá y quieres promocionar blanqueamiento dental. Una landing page enfocada solo en ese servicio convertirá mejor que tu sitio web general.

**2. Vas a hacer publicidad pagada (Google Ads, Facebook Ads)**
Cuando pagas por clic, quieres que el usuario llegue a una página diseñada específicamente para convertir, no a tu homepage donde se pierde.

**3. Tienes un objetivo de conversión único**
- Captar leads para curso online
- Vender tickets para evento
- Agendar citas para consultorio
- Descargar ebook o recurso

**4. Estás empezando y tienes presupuesto limitado**
Una landing page profesional ($1.2M - $2.5M COP) es más accesible que un sitio completo ($4M - $8M COP).

### Ejemplos de negocios colombianos que necesitan landing page:

- **María, consultora de imagen**: Landing para su curso "Armario Cápsula" - captura emails y vende el curso
- **Restaurante de comida saludable**: Landing para promocionar nuevo menú de almuerzos ejecutivos
- **Abogado especialista**: Landing para servicio específico de "divorcios express en Bogotá"
- **Agencia de viajes**: Landing para paquete "San Andrés todo incluido - promoción febrero"

## Cuándo necesitas sitio web completo

### Necesitas sitio web si:

**1. Tu negocio tiene múltiples servicios distintos**
Ejemplo: Eres una agencia de marketing que ofrece SEO, redes sociales, diseño web y publicidad pagada. Necesitas páginas separadas para cada servicio.

**2. Quieres posicionarte en Google con múltiples keywords**
Un sitio con 10 páginas tiene 10 veces más oportunidades de aparecer en búsquedas que una landing page.

**3. Necesitas credibilidad institucional**
Si vendes servicios B2B de alto valor ($5M+ COP), los clientes querrán ver:
- Quiénes están detrás (página nosotros/equipo)
- Casos de éxito (portafolio)
- Blog con contenido técnico (demuestra expertise)
- Página de servicios detallada

**4. Tu contenido cambia frecuentemente**
Si tienes blog, actualizas portafolio regularmente, o tienes noticias/eventos, necesitas un CMS (sistema de gestión de contenido).

### Ejemplos de negocios colombianos que necesitan sitio web:

- **Constructora**: Múltiples proyectos inmobiliarios, necesitan galería de cada uno, ubicación, especificaciones
- **Clínica médica**: Diferentes especialidades, múltiples doctores, blog de salud
- **Software company**: Producto SaaS con múltiples features, página de pricing, documentación
- **Tienda de moda**: Catálogo de productos, categorías, filtros, proceso de checkout

## ¿Y si necesito ambos?

Muchos negocios exitosos en Colombia tienen ambos:

1. **Sitio web corporativo** como "base" de su presencia online
2. **Landing pages específicas** para campañas y servicios particulares

**Ejemplo real:**
- Sitio web: omarhernandezrey.com (este portafolio)
- Landing pages potenciales:
  - /landing/chatbot-restaurantes-bogota
  - /landing/desarrollo-ecommerce-colombia
  - /landing/seo-local-medellin

Cada landing page se enfoca en un servicio específico para una audiencia específica, mientras el sitio web general muestra todo mi trabajo.

## Mi recomendación según tu etapa

### Si estás empezando ($2M-5M COP presupuesto)
**Empieza con una landing page profesional.**
- Es más rápido lanzarla (1 semana vs 1 mes)
- Te permite validar tu oferta antes de invertir más
- Puedes agregar páginas después si crece

### Si ya tienes tracción ($6M-15M COP presupuesto)
**Inversion en sitio web completo.**
- Posicionamiento SEO a largo plazo
- Credibilidad para clientes grandes
- Escalabilidad para agregar servicios

### Si ya tienes sitio web y quieres crecer
**Añade landing pages específicas.**
- Una landing por cada servicio principal
- Landings para campañas de publicidad
- Landings para diferentes segmentos de clientes

## Errores comunes que debes evitar

### Error 1: Hacer un sitio web de 20 páginas cuando solo vendes 1 servicio
Si eres freelancer y solo haces "consultoría SEO", no necesitas 15 páginas. Una landing page killer es mejor que un sitio vacío.

### Error 2: Hacer landing pages para todo sin tener sitio web base
Si alguien busca tu empresa en Google y solo encuentra landings de campañas antiguas, parece poco profesional.

### Error 3: Pensar que "más páginas = mejor"
Un sitio de 50 páginas con contenido mediocre es peor que uno de 5 páginas con contenido excelente.

## Precios actualizados para Colombia 2026

### Landing page profesional
- **Básica** ($1.2M COP): Diseño responsive, formulario, velocidad optimizada
- **Con A/B testing** ($2M COP): 2 variantes de diseño, integración analytics
- **Con chatbot** ($2.5M COP): Todo lo anterior + chatbot IA integrado

### Sitio web completo
- **Corporativo 5 páginas** ($4M COP): Inicio, nosotros, servicios, portafolio, contacto
- **Con blog** ($6M COP): Todo lo anterior + sistema de blog funcional
- **E-commerce básico** ($8M COP): Catálogo + carrito + pagos
