# Proceso: Manejo de Objeciones (Ventas)

**Objetivo:** Convertir el "está muy caro" o "lo voy a pensar" en un cierre.

## Prerrequisitos
- Haber leído `objections.ts`.

## Pasos
1. **Validar:** Nunca contradigas al cliente. Di "Entiendo perfectamente tu preocupación por X".
2. **Reencuadrar:** Muestra el costo de NO resolver el problema.
3. **Prueba Social:** Menciona un proyecto similar donde el cliente recuperó la inversión.
4. **CTA:** Ofrece un paso pequeño (ej. "empecemos solo con la landing y escalamos").

## Errores Comunes
- Bajar el precio de inmediato sin defender el valor.
- Tomarse la objeción como algo personal.
