# Proceso: Onboarding de Nuevo Cliente

**Objetivo:** Guiar al cliente desde el primer pago hasta el inicio del desarrollo sin fricción.

## Prerrequisitos
- Pago del anticipo (50%) confirmado.
- Formulario de onboarding completado por el cliente.

## Pasos
1. **Verificar datos:** Revisar en el Panel Admin que el brief esté claro. Si falta info, no empezar.
2. **Crear Slack/WhatsApp Group:** Unificar comunicación.
3. **Setear Repositorio:** Crear repo privado en GitHub y añadir al cliente (si es técnico).
4. **Agendar Kickoff:** Llamada de 15 min para alinear expectativas.

## Errores Comunes
- Empezar sin el anticipo (ERROR CRÍTICO).
- No definir el canal oficial de comunicación.
